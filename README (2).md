# 🛡️ AOFR Security Toolkit

> **Toolkit de Segurança Defensiva — AOFR TECH**  
> Desenvolvido por **Alfredo Ociola Francisco Romano** · Luanda, Angola

Colecção de ferramentas Python para auditoria de segurança defensiva, criptografia aplicada e reconhecimento OSINT. Criado para profissionais de segurança, auditores certificados e estudantes de cibersegurança.

---

## 📦 Ferramentas

| Ficheiro | Descrição | Uso Principal |
|---|---|---|
| [`cripto_aofr.py`](cripto_aofr/) | Toolkit AES & RSA com 14 módulos | Criptografia aplicada e educacional |
| [`audit_credenciais.py`](audit_credenciais/) | Auditoria de credenciais padrão em 300+ dispositivos | Pentesting de rede autorizado |
| [`dorks_audit.py`](dorks_audit/) | 1400+ Dorks Shodan & Google em 20 categorias | Reconhecimento OSINT defensivo |
| [`aofr_tech_audit.py`](aofr_tech_audit/) | Auditoria defensiva completa da infra AOFR TECH | Monitorização contínua de segurança |

## 🚀 Início Rápido

### Pré-requisitos

- Python 3.8+
- pip

### Instalação das dependências globais

```bash
git clone https://github.com/<teu-utilizador>/aofr-security-toolkit.git
cd aofr-security-toolkit
pip install pycryptodome PyJWT requests paramiko
```
